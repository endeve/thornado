Quick Start

  Clone

git clone https://github.com/endeve/thornado.git

   Setup your environment

export THORNADO_DIR=/path/to/thornado
export THORNADO_MACHINE=unique_user_label
Create a file in $THORNADO_DIR/Build/Machines to point to your HDF5 and LAPACK

  Compile

Problems are compiled in their own directory, e.g.,
cd $THORNADO_DIR/Sandbox/dgExperiments_Euler_NonRelativistic_IDEAL/Executables
make

   Run

Executable names being with "ApplicationDriver" and are suffixed with your $THORNADO_MACHINE environment variable.
./ApplicationDriver_Frodo
